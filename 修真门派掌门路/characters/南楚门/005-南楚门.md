# 南楚门

## 基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 南楚门 |
| 类型 | 修真门派 |
| 等级 | 元婴（楚老祖） |
| 所在 | 南疆 |
| 首次出现 | 第1卷 |

## 势力概况

### 宗门地位
- **南疆势力**: 在南疆拥有大量领土
- **楚老祖**: 元婴期，具体修为未详
- **与齐云派关系**: 同属楚家，但已独立

### 领地
- **黑河**: 楚秦门领地，东大门
- **九三坊**: 楚佑闵势力范围
- **南楚城**: 楚庄媛所在地

### 收留楚秦门原因
- 楚秦门是楚震当年帮助秦烈儿创立
- 传承有序，表面是独立宗门
- 需要一个附庸隔断与御兽门的人情往来
- 南楚门人丁少，对黑河"食之无味"

## 组织结构

### 楚家
- **楚老祖**: 元婴期
- **楚庄媛**: 筑基女修
- **楚佑闵**: 楚家第九房老三，九三坊坊主
- **楚佑严**: 齐云派楚家修士

## 势力关系

### 友好势力
- **楚秦门**: 被庇护的门派

### 敌对势力
- **御兽门**: 接壤，常有人情往来

## 主要事迹

- 派楚庄媛接楚秦门至黑河峰
- 楚秦门在黑河峰建立山门
- 楚佑闵对楚秦门态度冷淡

## 备注

南楚门是楚秦门的新靠山，虽有人情往来，但关系并不亲近。

## 势力范围归属

- **上级势力**: 南疆（御兽门势力范围）
- **同级势力**: 御兽门
- **下级势力**: 楚秦门（原附属）
