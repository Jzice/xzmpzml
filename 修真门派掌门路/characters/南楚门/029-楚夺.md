# 楚夺

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 楚夺 |
| 性别 | 男 |
| 修为 | 金丹 |
| 势力 | 南楚门 |
| 首次出现 | 第3卷 |
| 身份 | 南楚门长老、楚庄媛的师叔 |

## 角色关系

- 辈分：楚庄媛的师叔
- 职责：负责南楚门移民之事

## 主要事迹

- **总领移民事务**：南楚门近年来有大动作引入异姓修士，楚夺总领此事
- **斥责楚佑闵**：听闻楚秦门与楚佑闵家的纠纷后，斥责楚佑闵"和御兽门处不好，和齐云派处不好，现在和一帮练气底层的废物也处不好"
- **赐下玉简**：写了一道玉简给齐休，让其交给楚佑闵解决问题
- **威压震慑**：对齐休施放金丹修士威压，令其瑟瑟发抖

## 角色特点

- 头戴高冠的中年黑衣男修
- 面部轮廓深邃冷峻，鹰钩鼻
- 狭窄的双眼看人时如毒蛇噬骨
- 气质孤傲凶狠，阴鹫
- 对楚庄媛较为宽容

## 备注

- 将齐休贴上"废物"的标签
- 曾把齐休和楚庄媛用袍袖扫出殿外
- 是解决楚秦门与楚佑闵家矛盾的关键人物
