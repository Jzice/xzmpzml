# 齐休

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 齐休 |
| 性别 | 男 |
| 修为 | 练气二层 → 金丹 → 元婴（最终） |
| 本命 | 单本命（本命乃无数世界难寻之物，但于大道无缘） |
| 同参 | 指猴（一阶灵兽，黑白相间，人手指般大小） |
| 势力 | 楚秦门（第五代掌门） |
| 首次出现 | 第1卷 |

## 角色关系

- 师承：齐掌门（楚秦门第三代掌门，恩若父子）
- 门人：张世石、展元、何玉、古吉、秦唯喻、黄和、虞景、沈昌、潘荣
- 前任：秦斯言（第四代掌门，入赘安家）

## 主要事迹

- 三岁时被验出极品修仙体质，但本命无物可同参，断绝大道希望
- 被师父派到偏僻处十几年，暗中为掌门贩卖物资换取筑基丹
- 29岁师父去世，意外成为第五代掌门
- 带领楚秦门南迁南疆

## 备注

齐休是本书主角，从练气小修士成长为元婴大能，一手将楚秦门从濒临灭门的小派发展成白山势力。
