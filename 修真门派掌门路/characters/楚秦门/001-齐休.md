# 齐休

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 齐休 |
| 性别 | 男 |
| 修为 | 练气三层（已进阶）→ 金丹 → 元婴（最终） |
| 本命 | 【赤尻马猴】（废本命） |
| 同参 | 玲珑塔（一阶法器，诡代修行用） |
| 势力 | 楚秦门（第五代掌门） |
| 首次出现 | 第1卷 |

## 角色关系

- 师承：齐掌门（楚秦门第三代掌门，恩若父子）
- 门人：张世石、展元、何玉、古吉、秦唯喻、黄和、虞景、沈昌、潘荣、余德诺
- 前任：秦斯言（第四代掌门，入赘安家）

## 主要事迹

- **本命天赋觉醒**：
  - 【不在算中】：无法被任何命运之术预测、演算；记忆不能被任何方式夺取；灵魂不能被任何法术控制
  - 【明己心】：运转此天赋时万碍不侵，直明本心，修行中可省无数艰难险阻
- **获得玲珑塔**：南楚门金丹楚夺所赐
- **修炼诡代之法**：以【玲珑塔】为同参，修炼楚慧心改写的【明心见性诀】，进阶练气三层
- **被迫参与秘密**：被楚夺要求参与偷盗婴儿送入南楚门的秘密任务
- **建立黑河坊**：与赵良德、王涫、楚佑闵结成利益共同体
- **经历楚红裳之怒**：目睹白晓生被钉在坊市示众

## 诡代修行原理

1. 诡代的本命：与真本命有共通之处的【七窍玲珑心】
2. 以【玲珑塔】作为同参之物
3. 修炼诡代出来的假本命
4. 通过假本命和真本命的共通特性，间接增进修为

## 备注

齐休是本书主角，从练气小修士成长为元婴大能，一手将楚秦门从濒临灭门的小派发展成白山势力。
