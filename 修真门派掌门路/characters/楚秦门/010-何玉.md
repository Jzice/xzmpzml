# 何玉

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 何玉 |
| 性别 | 男 |
| 年龄 | 17岁（第4卷） |
| 修为 | 练气五层 → 练气六层（冲击中） |
| 灵根 | 水土双灵根 |

| 同参 | 流水菩提串 |

## 角色关系

- 导师：阚林（筑基散修，常来指点修行）
- 同门：张世石（对其颇为照顾）

## 主要事迹

- **黑河蜥之战**：领悟战斗类本命天赋【冰棱地刺】
- **修为飞涨**：一年从练气三层进阶到练气五层
- **冲击六层**：正在冲击练气第二槛（练气六层）
- **性格变化**：随修为增长越发傲气，与同门渐行渐远
- **参与擂台赛**：代表楚秦门参加黑河坊争夺擂台赛

## 本命天赋

- **【冰棱地刺】**：战斗类本命天赋（黑河蜥之战领悟）

## 角色特点

- 十七岁时个头蹿高，越发丰神俊朗
- 顾盼自豪，远近有名的青年修士
- 修炼极为刻苦，阚林十分看好其前途
- 对阚林感情深厚，楚红裳事件中见阚林受伤十分心痛

## 备注

何玉是楚秦门最有希望筑基的修士，后成为楚秦门重要战力。
