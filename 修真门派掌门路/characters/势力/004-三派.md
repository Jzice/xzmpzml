# 三派（流云宗、荷花观、湟左詹家）

## 基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 流云宗、荷花观、湟左詹家 |
| 类型 | 修真门派 |
| 等级 | 筑基门派（各有筑基修士） |
| 所在 | 齐云派范围，楚秦门周边 |
| 首次出现 | 第1卷 |

## 势力概况

### 三派背景
- **流云宗**: 三派之一
- **荷花观**: 三派之一，有筑基女修
- **湟左詹家**: 三派之一

### 联合原因
- 楚秦门是金丹老祖创下的基业
- 山门内灵地灵田数量和质量都超过三派之和
- 三派联合吞并楚秦门

### 内部安排
- 三派约定吞并后合而为一
- 已将自家基业送人，换取其他宗门不介入
- 不达目的誓不罢休

## 组织结构

### 各派掌门
- **流云宗**: 有筑基修士参与吞并
- **荷花观**: 何观主（筑基女修）
- **湟左詹家**: 家主（筑基修士）

### 代表人物
- 流云宗男修: 夺得新任"掌门"之位
- 荷花观女修: 泼辣，参与守门
- 湟后詹家: 安七代表家族参与

## 势力关系

### 与楚秦门关系
- **敌对**: 吞并楚秦门的元凶
- 三派联手，关闭护山大阵，杀上主峰

### 与齐云派关系
- 同属齐云派门下
- 道门之间讲香火情，较忌讳闹出人命
- 但为了利益仍铤而走险

## 主要事迹

- 趁齐掌门病重，元气大伤，寿元只剩三年时发起吞并
- 三派联合，设下比武选拔"掌门"的闹剧
- 楚佑严代表楚家出面，三派被迫允许楚秦门南迁
- 签订契约：楚秦门不得报仇，三派不得再为难

## 备注

三派是楚秦门的直接敌人，趁人之危吞并小派，展现了修真界的残酷生存法则。

## 势力范围归属

- **上级势力**: 齐云派
- **同级势力**: 姜家、刑家、安家
- **下级势力**: 无
