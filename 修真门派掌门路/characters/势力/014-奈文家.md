# 奈文家

## 势力概览

| 项目 | 内容 |
|------|------|
| 势力名称 | 奈文家 |
| 类型 | 修真家族 |
| 所在地 | 罗家旧地（已覆灭） |
| 势力特点 | 病气功法 |
| 主要人物 | 奈文落、奈文霖 |
| 结局 | 被楚秦门灭门 |

## 势力历史

### 罗家旧地鼎立

- 奈文家与宁家、敢家在罗家旧地三足鼎立
- 奈文家以病气和病火功法闻名
- 家主奈文落是筑基高阶修士

### 与楚秦门的冲突

- 奈文家与宁家合攻楚秦山
- 楚秦山之战中奈文家被灭门
- 奈文落战死
- 家族修士全部慷慨赴死

## 主要成员

### 奈文落

- 奈文家家主
- 筑基高阶
- 病火功法
- 战死于楚秦山

### 奈文霖

- 奈文家掌上明珠
- 修行病魔道
- 与赵瑶有特殊关系
- 为救齐休牺牲自己

## 备注

- 奈文家以病气立家
- 家族覆灭时展现出不屈精神
- 临死唱诵家族誓言
