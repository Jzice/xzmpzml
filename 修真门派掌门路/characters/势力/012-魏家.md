# 势力档案-魏家

## 基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 魏家/魏玄宗门 |
| 类型 | 修真宗门 |
| 所在 | 白山（山都山） |
| 首次出现 | 第5卷 |

## 势力概况

原为北方小宗门，由魏玄主持，后迁移到白山。魏玄是金丹修士，与魏同（御兽门金丹老祖）为远房亲族。魏同借助御兽门的势力，召集各路修士灭了山都门，将地让给魏玄作为新的宗门所在。

## 组织结构

- 魏玄 - 金丹修士，新宗门主人
- 魏同 - 金丹修士，御兽门南疆分支，寿元无多
- 魏敏（敏娘）- 魏玄后辈，丧夫带女，被迫嫁给齐休

## 势力关系

- 盟友：南楚门（楚夺）
- 对立：山都门（已灭）

## 主要事迹

- 第5卷灭山都门之战
- 魏玄迁移宗门至白山
- 魏敏被迫嫁给齐休

## 备注

魏玄与魏同是远房亲族，魏同出力，魏玄出名，两家并一家。

## 势力范围归属

- **上级势力**: 白山
- **同级势力**: 器符盟、山都门（原敌对）、丹盟、灵木盟
- **下级势力**: 无

## 设定关联

- 白山战争
- 山都门覆灭
