# 敢家

## 势力概览

| 项目 | 内容 |
|------|------|
| 势力名称 | 敢家 |
| 类型 | 修真家族 |
| 所在地 | 罗家旧地 |
| 主要人物 | 敢毅、敢珑 |

## 势力历史

### 罗家旧地鼎立

- 敢家与宁家、奈文家在罗家旧地三足鼎立
- 家主敢毅是筑基修士
- 敢家在罗家旧地北部有影响力

### 战后格局

- 奈文家覆灭后，敢毅获得空曲山
- 敢毅想让梨山给楚秦门，齐休推辞
- 敢毅与敢珑分家
- 敢毅为正式修真家族
- 敢珑只能算散修修真家族

## 主要成员

### 敢毅

- 敢家家主
- 筑基修士
- 与楚秦门联合对抗奈文家

### 敢珑

- 敢家成员
- 敢毅分家后做梨山之主
- 散修修真家族

## 备注

- 战后仰仗楚秦门鼻息
