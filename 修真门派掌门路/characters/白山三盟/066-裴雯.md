# 裴雯

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 裴雯 |
| 性别 | 女 |
| 修为 | 元婴（较弱） |
| 本命 | 水之润泽 |
| 势力 | 白山（敌对楚红裳） |
| 首次出现 | 第14卷 |

## 角色关系

- **身份**: 楚红裳前闺蜜，后反目成仇
- **敌人**: 楚红裳、楚神通
- **盟友**: 连水盟、灵木盟、离火盟、庶务峰三家

## 主要事迹

### 与楚红裳的恩怨

- 裴雯是楚红裳曾经的闺蜜
- 知道楚红裳的真实本命：【九天不灭炼火霓裳】，修火之不灭大道
- 楚红裳隐藏真实根脚，裴雯对此耿耿于怀
- 两人反目成仇

### 黑河坊之战

- 楚夺死后，裴雯现身对付楚红裳
- 布置【镇灭彩光大阵】，专门克制楚红裳
- 七彩虹桥罩住楚家所有人
- 联合白山三盟和庶务峰燕、胡、刘三位元婴
- 联手攻击楚红裳和楚神通

### 被姬信隆阻止

- 姬信隆突然现身阻止
- 姬信隆是大周书院修士，收了楚红裳和齐休的贿赂
- 裴雯等人被姬信隆状告
- 大周书院化神介入，要求裴雯等人先散了
- 裴雯的复仇计划失败

### 威胁楚红裳

- 临走前指着楚红裳破口大骂
- "逃得了初一，你也逃不了十五"

## 备注

- 裴雯修为在元婴中是最弱的一批
- 水之润泽真意不利于争斗
- 为复仇花费大量心思
