## 大纲

> 楚秦与离火达成合议，山都守军投降楚秦，楚秦则继续南下取灵木的博木城。乐川攻下醒狮谷狮巢后身死。楚秦在楚问、楚神通出关后攻下博木城，命名“楚恩城”。沙诺在外海被裴家两位修士追杀，逃至魔井，却遇见元婴魔蛇，被困井底。楚秦计划并盟为门，齐休闭关五十年，进阶金丹八层，期间灵木趁楚问回海楚城、楚神通去齐云派和青莲剑宗等各家联军对峙前线又来扰，在楚神通回来时退却。楚红裳出关，渡给齐休一道不灭生机，楚神通与蔡渊协调好齐休在楚云峰结婴之事，欲拱齐休结婴后争齐云掌门。姜炎结婴前夕返回白山了结因果执念，先寻南宫嫣然，致后者心火郁结而亡，后又寻姜明恪，最后来寻齐休，被齐休逃过一死，反而激起齐休一心向道元婴。楚无影修至金丹圆满，去蓝凤异蝶老巢结婴，发现诡代之法结婴存在问题，计划回白山告知齐休。齐休这边处理完宗门事务开始在楚云峰闭死关冲击金丹圆满，闭关中屡遭公正之鬼侵扰。外海一行人寻回沙诺，而沙诺已堕入魔道。楚无影借白山新晋化神的天地之威体悟到本命、诡代之法、大道三者联系，不料被御兽门抓住，领悟到影遁后逃离，齐休听闻后出关欲援救楚无影，与众人共同前往九星坊，后楚无影在众元婴包围下被俘。楚无影为提醒齐休诡代之法结婴的问题，强行结婴引发天地异象，被冯甄出手搜魂硬断，而后被白山化神劫走，终活下来。
