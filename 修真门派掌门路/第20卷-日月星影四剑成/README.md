## 大纲

> 楚秦门派去白山各大宗门奸细被拔。万轩告知齐休碧湖宫底密藏消息，双楚与楚秦盟组成老年团去密藏探宝，不料后面出现“黑手”搅局，杀人并抢夺长钉物事。事发后，各地黑手老巢被端，楚无影加入黑手事发，潜逃醒狮谷。齐休被欲再寻何玉的南宫家软禁，期间齐云姜家允其丰厚条件，遂答应由顾叹主导帮其覆灭碧湖门。展剑锋前去酆水参加开辟战争，期间筑基成功。莫剑心以身饲剑，筑出日、月、星、影四把无形剑。
