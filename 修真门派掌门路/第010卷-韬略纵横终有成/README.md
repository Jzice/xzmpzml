# 第010卷-韬略纵横终有成

## 摘要

楚秦门于罗家旧地夺取剑影山，改名楚秦山来作安身之地，而后暗定三分之计，先拱卫宁家，后投靠敢家，终使宁家、奈文家、敢家于罗家旧地三足并立。
