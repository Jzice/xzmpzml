## 势力概况

白山是此界最顶级的超级宗门之一，与齐云、御兽门、大周书院等并列。白山势力由白山山顶和分布在白山区域的修真势力组成。白山之主贾长庚拥有转世轮回的特殊功法，其修为深不可测。更重要的是，白山顶上可能拥有关于此界真相的最大秘密，这一秘密关系到此界的根本起源与运转机理。白山作为顶级势力，其影响力辐射整个白山区域，区域内大小宗门、家族、联盟皆以白山马首是瞻。白山内部势力错综复杂，既有白山顶的直接掌控，也有白山五行盟这样的联盟体系，旗下还有众多小型势力。
