## 势力概况

奈文家是罗家旧地的修真家族，与宁家、敢家在罗家旧地三足鼎立。奈文家以病气和病火功法闻名，家主奈文落是筑基高阶修士。

奈文家与宁家合攻楚秦山，在楚秦山之战中被灭门。奈文落战死，家族修士全部慷慨赴死，临死唱诵家族誓言，展现出不屈精神。
