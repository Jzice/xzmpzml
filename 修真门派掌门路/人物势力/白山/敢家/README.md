## 势力概况

敢家是罗家旧地的修真家族，与宁家、奈文家在罗家旧地三足鼎立。家主敢毅是筑基修士，敢家在罗家旧地北部有影响力。

奈文家覆灭后，敢毅获得空曲山。曾想让梨山给楚秦门，但齐休推辞。后来敢毅与敢珑分家，敢毅成为正式修真家族，敢珑只能算散修修真家族。战后敢家仰仗楚秦门鼻息。
