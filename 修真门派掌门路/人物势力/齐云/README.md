# 齐云派

## 简介

齐云派是本界的超级宗门。齐云派境内有五座修真城市，分别为滕老祖下辖的齐东城、

齐云派明面最高修为就是化神修士，再往上也有，但不会在此界呆了。

齐东、齐云、齐南城各一位，负责五行灵根修士教习管理的五峰座主一位，负责其他灵根修士的传功峰座主一位，天地峰座主一位，庶务峰座主一位，执法峰座主一位，还有位秘境座主，守护进阶化神必须的七阶灵地，共九位化神修士。[1]

## 势力概况

### 宗门地位
- **顶级势力**: 至少拥有9位化神期修士的超级宗门
- **统治范围**: 齐云国（侯国）
- **治下宗门**: 大大小小的贵族和平民，以及附属修真门派

### 宗法制
- 凡人与修士分离，"老死不相往来"
- 门下宗门三代之内受保护，三代之后可自由攻伐
- 化神期可授予侯爵

## 组织结构

### 势力关系
- **楚秦门**: 秦烈儿是楚震亲传弟子
- 楚秦门开山时，"楚"字在前以示感恩

## 势力关系

### 附属势力

- **南宫家**: 齐云派化神修真家族，拥有一位化神：南宫木
- **陆家**: 齐云派掌门陆云子所在的元婴修真家族
- **楚家**: 齐云派长老楚震的修真家族，是主角所在楚秦门的上级门派

### 重要人物

- 天地峰座主，姓名、职务不明，本界第一档次的化神。三楚、齐云历代掌门的主家。疑似原双山宗成员，与南林寺枯荣、白山化神贾长庚为师兄弟。
- 刑铣，执法峰座主


## 主要事迹

- 天理门、青莲剑宗、稷下城、齐云派等势力在碧湖秘境等待底层开放，却被全知神宫逃至死亡沼泽。于是在周围开始大肆搜寻。[2]
- 因天地峰座主打走聂疯子，打伤枯荣一事，与各超级宗门交恶对峙。
- 天地峰座主宣布闭死关，很多人以田尝马首是瞻。天地峰一系也积极向田家靠拢。[3]


## 备注

齐云派是本书中齐云国范围内的最高修真势力，统治着无数小门派。
