## 势力概况

齐南城是齐云派在南方的重要据点，作为齐云派南方的大城，齐南城承担着齐云势力范围南部的政治、经济、军事中心职能。城池地理位置优越，是齐云派向南方扩展势力、辐射影响的重要节点。

齐南城与齐云城同为齐云派的核心城池，但侧重点有所不同。齐南城更偏向于对外交流、南方防御以及与周边势力的联络。城中设有多个情报机构和军事设施，是齐云派维持南方统治的重要支柱。

## 势力范围归属

- **上级势力**: 齐云派
- **同级势力**: 齐云城、执法峰、兵站坊
- **相关势力**: 楚秦门、南楚门
