## 势力概况

南楚门是楚家门下的分支门派，位于南疆，拥有大量领土。南楚门与齐云派同属楚家，但已独立成派。楚家是齐云派的核心家族，元婴后期的楚震（楚老祖）威震齐云近两千年，南楚门虽与齐云派同属楚家，但已独立运作。

### 宗门地位

- **南疆势力**：在南疆拥有大量领土
- **楚老祖**：元婴期，具体修为未详
- **与齐云派关系**：同属楚家，但已独立

### 领地

- **黑河**：楚秦门领地，东大门
- **九三坊**：楚佑闵势力范围
- **南楚城**：楚庄媛所在地

### 收留楚秦门原因

- 楚秦门是楚震当年帮助秦烈儿创立
- 传承有序，表面是独立宗门
- 需要一个附庸隔断与御兽门的人情往来
- 南楚门人丁少，对黑河"食之无味"

## 势力范围归属

- **势力范围**: 南疆（御兽门势力范围）
- **上级势力**: 齐云楚家
- **下级势力**: 楚秦门（原附属）
- **同级势力**: 御兽门

## 主要事迹

- 派楚庄媛接楚秦门至黑河峰
- 楚秦门在黑河峰建立山门
- 楚佑闵对楚秦门态度冷淡
- 黑河坊之战中派遣援军
- 楚红裳带领南楚门六千人援军支援黑河坊
