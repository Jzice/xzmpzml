# 第019卷-白山外海显身手

## 摘要

白山大战继续，楚秦门趁机在外海与白山各大宗门之间埋钉子。齐休与刑剑交换情报得知连水门北迁的消息，为平衡白山局势，将此消息告诉了灵木盟柴艺，让他搅和此事。后又向齐云掌门陆云子讲述邢剑为抓捕姜炎借归儒势力介入南宫木独霸北丁申山一事，归古势力也牵扯其中，因此陆云子帮忙否掉连水门北迁之事。何欢宗下场围离火救丹盟，白山攻伐双方终决定议和。丹盟欠债甚多，齐休与何欢宗中行隽、丹盟内部原外海散修英伯等人谋划，终迫使丹盟决定卖家底还债。楚秦门联合英伯、外海势力覆灭安身外海的龙家。
