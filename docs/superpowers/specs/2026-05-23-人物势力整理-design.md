## 三、顶层 README 结构

### 3.1 内容组成

```
1. 势力范围总览（化神级势力）
2. 目录结构索引（反映实际目录）
3. 势力×角色汇总表（按势力分组展示所有角色）
4. 角色关系图（保留并更新）
```

### 3.2 势力×角色汇总表设计方案

按以下分类组织角色表：

**一、超级宗门（化神级）**
- 齐云 → 列出齐云山峰/家族/城市/商会/附属的角色
- 御兽门 → 列出御兽门总山/摩云城/铁风群岛等的角色
- 白山（白山顶） → 白山顶相关
- 大周书院 → 大周书院角色
- 南林寺、青莲剑宗、天理门 → 各自角色

**二、白山区域势力**
- 楚秦门 → 楚秦门全体角色
- 白山五行盟 → 灵木/连水/离火/锐金/厚土
- 白沙帮、山都门、白山剑派等
- 何欢宗、幻剑盟、丹盟、器符盟等

**三、齐云附属势力**
- 南楚门
- 安家、姜家、裴家等家族
- 兵站坊等机构

**四、外海势力**
- 龙家、海楚门、明家、海东城

**五、其他独立势力**
- 黑风谷、稷下城、阳明山、蛮荒/醒狮谷

**六、散修**

### 3.3 角色表格式

每个角色表包含：姓名、修为、身份/备注。

数据来源：从各势力目录下的角色档案文件自动汇总。
