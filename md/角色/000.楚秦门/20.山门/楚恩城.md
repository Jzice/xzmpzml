# 楚恩城

## 基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 楚恩城（原名博木城） |
| 类型 | 修真仙城 |
| 位置 | 白山南部，灵木盟原龙兴之地 |
| 灵地品阶 | 四阶（主峰约可容纳一人修行的四阶灵地） |
| 护山大阵 | 四阶上清灵宝大阵（购自万宝阁） |
| 首次出现 | 第639章（以博木城之名） |

## 历史沿革

### 原名博木城

博木城原为灵木盟龙兴之地，乃灵木盟起家的根基仙城。后因白山深处开辟，灵木盟将重心转向博森城，博木城逐渐没落，远不如连水、离火两城繁盛。

### 山都之战后易主

第023卷至第024卷，楚秦与离火达成合议后，三楚（齐云楚家、南楚门、海楚门）与楚秦联军南下攻打博木城。在楚神通、楚问两位元婴老祖出关后，联军于第643章攻破博木城。灵木盟在离火大军和楚神通的双重压力下放弃死守，退走前将城中设施破坏殆尽。

### 改名楚恩城

攻占后，齐休为表明永世不忘楚家恩情，将博木城改名为**楚恩城**，并亲笔题写"楚恩仙山"四字，命广汇阁刻好立于山门。又重建北城门，命名"楚恩门"。

### 护山大阵

齐休从万宝阁购买了一套四阶【上清灵宝大阵】（齐云跟脚，道门正宗），为楚恩城提供安全保障。

### 股权结构

- 三楚（齐云楚家、南楚门、海楚门）：六成
- 楚秦门：两成
- 青丹门：两成

### 传送法阵

后山地底有传送法阵可通齐云，由楚问命族中修士帮忙维修，以便有外敌时齐云方面可快速支援。

## 战略地位

楚恩城位于白山南部，是楚秦盟南方的战略前哨。齐休曾誓言"为楚家守好这南大门"，楚恩城在后续的楚秦与灵木、厚土、锐金等势力的战争中发挥了重要的南方基地作用。

## 重要事件

### 楚恩城大典（第652章）

楚恩城大典当日，齐休因在齐云处理结婴事宜未归，顾叹宣布由秦长风作为楚恩城实际城主主持大典。齐云派楚佐笙、陆家、万宝阁、蓝家等多位金丹修士到场，另有青丹门英伯、江南宗姜明恪、厚土盟秘密使者等各方势力人物云集。法引禅师提前到达，与展剑锋、宋仲谦等旧交相谈甚欢。

### 姜炎偷袭（第654章）

姜炎结婴前夕返回白山了结因果执念，在楚恩山同参殿偷袭齐休。齐休在万骨鬼修辅助下逃过一死，此事反而激起齐休一心向道冲击元婴的决心。

### 并盟为门基地

楚恩城也是楚秦门从楚秦盟并盟为门过程中的关键基地。齐休在城中闭关五十年，进阶金丹八层。楚秦盟高阶修士因拥有此城，灵地分散问题迎刃而解。

## 代表人物

- **齐休**：楚秦门掌门，楚恩城的命名者和守护者
- **秦长风**：楚恩城实际上的城主，负责城内交通内外
- **顾叹**：楚秦谋主，掌盟务，协助治理楚恩城
- **南宫嫣然**：秦长风之妻，负责城中庶务

## 势力范围归属

- **所属势力**：楚秦门 / 楚秦盟
- **上级关联**：三楚（齐云楚家、南楚门、海楚门）拥有六成股份
- **同级关联**：青丹门拥有两成股份
- **地理位置**：白山南部，原灵木盟领地

## 设定关联

- **灵木盟**：楚恩城原为灵木盟龙兴之地博木城，灵木盟失去此城后实力大损
- **楚秦门扩张**：楚恩城是楚秦门从白山中部势力向南方扩张的里程碑
- **三楚联盟**：楚恩城的股权结构体现了三楚与楚秦的联盟关系
- **上清灵宝大阵**：四阶道门正宗护山大阵，购自万宝阁
- **白山战争**：楚恩城在第024-025卷的白山大战中是楚秦盟在南方的战略支点

## 备注

- 楚恩城命名中的"楚恩"意为永志楚家恩情，是齐休刻意将"秦"字去掉的政治信号
- 城中四阶灵地不大，约相当于楚云峰五阶灵地的格局，仅可容纳一座草堂、一人修行
- 楚恩城是楚秦门拥有的第一座仙城级别的山门
