## 势力概况

阳明山是此界的儒家宗门，属儒门唯心一脉。阳明山境内有完整的官府、差役、律令、刑罚体系，注重教化，一向排外。对道门等外来势力不甚友善。

阳明山与天理门、稷下城等儒门势力同属归儒派阵营，曾是正气坊正道联军的一部分，与黑风谷形成化神级别的对峙。

在地缘上，阳明山位于齐云派西北端，与南林寺、齐云派三家交界处即为稷下城所在。
