## 势力概况

稷下城是此界其他势力之一。据现有记载，稷下城以道德碑闻名，其道德碑与张世石的无名功德碑极为相似。稷下城势力与道德碑信仰体系紧密相连，是此界重要的功德碑传承之地。势力具体情况待进一步查证。
