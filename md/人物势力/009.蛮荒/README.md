## 势力概况

蛮荒是此界的一个重要地域概念，涵盖广大的未开发区域和凶险之地。蛮荒之中生存着大量高阶古兽、灵兽和凶兽，是人类修士极少涉足的危险地带。其中以醒狮谷最为著名，是化神元犼"醒狮"的领地。

### 主要区域

| 区域 | 类型 | 说明 |
|------|------|------|
| 醒狮谷 | 蛮荒凶地 | 化神元犼老狮子的核心领地 |
| 蛮牛荒原 | 蛮荒荒原 | 化神古兽"蛮牛"的领地，老狮子南迁后的去向 |
| 摩云山 | 古兽领地 | 摩云鬣把守的峡谷，后建为摩云城 |
