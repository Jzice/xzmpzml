## 势力概况

醒狮谷是此界著名的蛮荒凶地，因化神元犼"醒狮"（老狮子）而得名。谷内生存着大量古兽、灵兽、凶兽，是老狮子的核心领地。大周书院称老狮子是"与人类接壤的蛮荒里已知最强大的存在之一"。

醒狮谷地势复杂，从北到南依次分布着摩云鬣把守的峡谷入口、四翅天鹰的河川峡谷、人面纹蛇的蛇道洞穴、风息归土兽的黑土地（重土之地），以及最深处老狮子所在的峡谷。

### 历史事件

- **醒狮谷得名**: 因化神元犼醒狮的领地而得名
- **血刀藏身**: 血影邪修血刀曾逃入醒狮谷，在血池附近修炼血影邪功
- **楚秦门复仇**: 齐休带队进入醒狮谷，为展元报仇，联手击杀了血刀
- **展仇陨落**: 展仇在重土地底与北丁申山鬼修同归于尽
- **御兽门开辟战争**: 御兽门试图开辟醒狮谷，与老狮子爆发大战，损失惨重
- **众古兽南迁**: 老狮子与蛮牛大战后，醒狮谷众古兽集体南下蛮牛荒原

### 主要地点

- **摩云山/摩云崖**: 摩云鬣把守的峡谷入口，后建为摩云城
- **河川峡谷**: 四翅天鹰的领地
- **蛇道洞穴**: 人面纹蛇的巢穴
- **黑土地/重土之地**: 风息归土兽的领地
- **血池**: 血刀修炼血影邪功的地方
- **老狮子峡谷**: 化神醒狮所在的禁地
