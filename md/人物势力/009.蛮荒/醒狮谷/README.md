## 势力概况

醒狮谷是白山附近的一个修真势力，以御兽术闻名。谷内设有血池，是血影邪修的修炼之地。

### 历史事件

- **血刀藏身**: 血影邪修血刀曾逃入醒狮谷，在血池附近修炼血影邪功
- **楚秦门复仇**: 齐休带队进入醒狮谷，为展元报仇，联手击杀了血刀

### 主要地点

- **血池**: 血刀修炼血影邪功的地方，位于醒狮谷深处
