## 势力概况

南楚是对齐云楚家在南疆地区发展壮大的势力的称呼，也可视为楚家在南疆的别称。楚家作为齐云派的核心家族，其成员在南疆开拓立宗，逐渐形成了独立的势力分支。

### 起源与发展

- **楚震时期**：楚震收秦烈儿为徒，助其创立楚秦门，开启了楚家对外发展的序幕
- **楚红裳时期**：楚红裳在南疆开拓立宗，成为南楚门实际掌控者之一
- **楚神通时期**：楚神通之子楚问创立海楚门

### 与楚家关系

南楚与齐云楚家本为一体，只是因地理位置和发展历程不同，形成了不同的势力分支：

- **齐云楚家**：位于齐云派核心区域，由楚震统领
- **南楚**：指南疆地区的楚家势力，由楚红裳、楚神通等统领
- **南楚门**：楚家门下的独立门派

## 势力范围归属

- **势力范围**: 南疆（御兽门势力范围）、齐云
- **上级势力**: 齐云派
- **下级势力**: 楚秦门、海楚门

## 主要事迹

### 第14卷 - 夺丹试炼与进阶

- 楚红裳在夺丹试炼中经历幻梦
- 在幻梦之中彻底放下当年情憾
- 补完道心，一举进阶元婴中期
- 进阶庆典在南楚城举办

### 第16卷 - 黑河坊之战

- 得知黑河坊被灵木盟围攻后率军来援
- 带领齐云楚家三千子弟及南楚门六千人援军
- 火云万丈，气势非凡
- 单挑龙家家主，将其一击打退
- 本命【九天不灭炼火霓裳】使她几乎打不死
- 多次俯冲攻击，号称"疯女人无敌"

### 思过山大战

- 救援楚秦门于危难之中
- 与楚无影、楚青玉等配合
- 阻止龙家家主追击楚秦门飞梭

## 与其他势力的关系

## 备注

1. "南楚"一词既可指南疆地区的楚家势力，也可指代楚家门下的南楚门
2. 楚红裳是南楚的标志性人物，号称"疯女人无敌"
3. 南楚与齐云楚家本为一体，共同代表楚家利益
4. 楚问创立海楚门，标志着南楚势力的进一步扩张
