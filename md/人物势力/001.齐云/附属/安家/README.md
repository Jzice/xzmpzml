## 势力概况

安家是齐云派的附属家族之一，势力隶属于齐云派管辖。安家与齐云核心家族楚家有从属关系，是齐云势力体系中的重要组成部分。

安家最著名的事件是秦斯言入赘安家，与安红儿结为道侣。秦斯言原为楚秦门第四代掌门，后入赘安家，成为安家之人。
