## 势力概况

庶务峰是齐云派的核心职能峰头之一，负责处理齐云派的日常事务管理。庶务峰由化神修士陈悫担任座主，陈氏家族繁衍兴盛，与田氏并称齐云两大顶级家族。在天地峰座主闭死关后，庶务峰与执法峰成为齐云派实际维持运转的核心力量。
