## 势力概况

楚家是齐云派的核心家族，拥有的元婴后期修士楚震（楚老祖）威震齐云近两千年。楚家虽拥有元婴修士的超级战力，但一直没有从齐云派中独立出来，始终作为齐云派的核心家族存在。

楚家与楚秦门渊源极深。当年楚震收秦烈儿为徒，助其创立楚秦门，"楚"字放"秦"字前面便是秦烈儿为示永世感念楚震恩情所为。数千年来，楚家虽多次表示不再管楚秦门之事，但在关键时刻仍会出手相助。

### 历史渊源

- **楚震时期**：楚震年近两千岁，已快到元婴修士大限，人越来越念旧
- **楚红裳时期**：楚红裳进阶元婴中期，成为南楚门实际掌控者之一，与楚神通共同代表齐云楚家利益

## 主要事迹

### 数千年前
- 楚震收秦烈儿为徒
- 秦烈儿因同门争斗在齐云派金丹修士中树敌过多，被齐云派逐出
- 楚震帮秦烈儿争取开拓机会、寻找帮手
- 为秦烈儿争取到楚秦山作为开宗立派之地
- 师徒情分至此了结

### 第1卷
- 楚震年近两千岁，已快到元婴修士大限
- 楚红裳在南疆开拓立宗，拜托楚震
- 楚震想起楚秦门快灭门之事
- 派楚佑严往楚秦山走一趟
- 保下楚秦门的宗门传承，送至南疆
- 明确表示楚家对楚秦门仁至义尽，不要自以为有特别交情

### 第14卷
- 楚红裳在夺丹试炼中进阶元婴中期
- 进阶庆典在南楚城举办

### 第16卷
- 黑河坊之战爆发
- 楚红裳率军来援，带领齐云楚家三千子弟及南楚门六千人援军
- 单挑龙家家主，将其一击打退

## 备注

1. 楚家虽然是拥有元婴修士的超级家族，但一直没有从齐云派中独立出来
2. 楚震虽已尽师父责任，但念及旧情，在寿元将近时仍出手保下楚秦门传承
3. 楚红裳是楚家在南楚门的代表，号称"疯女人无敌"
4. 楚家与南楚门关系密切，是齐云楚家在白山势力的重要触角
