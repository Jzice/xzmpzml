## 势力概况

外海是指御兽门势力范围外的海域地带，是相对于白山地域的广大海域。在御兽门的飞地摩云城之外，是更为广阔的海域世界。

外海地区势力复杂，各方势力在此角力。白山战争结束后，部分势力选择远迁外海寻求发展。

## 势力范围

- **主要势力**: 御兽门飞地摩云城
- **相关势力**: 海楚门、龙家（已覆灭）
- **涵盖区域**: 御兽门势力范围外的广大海域
