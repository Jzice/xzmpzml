## 势力概况

龙家是白山的一个修真家族，后因与归儒派勾结的阴谋败露，被迫远迁外海，最终覆灭。

龙家与灵木盟有所勾结，参与了进攻楚秦门的战争。龙越云作为龙家成员，担任与归儒派联络的中人角色。

## 主要事迹

### 归儒派联络

龙家与归儒派有所勾结，通过龙越云作为中人联络。楚秦门发现这一阴谋后出首揭发，导致龙家被迫迁往外海。

### 思过山大战

龙家参与了灵木盟进攻楚秦门的战争，是灵木盟的重要盟友。

### 龙家覆灭

龙越云被杀后，龙家与归儒派勾结的阴谋败露。楚秦门出首揭发，龙家被迫远迁外海，最终在外海覆灭。
