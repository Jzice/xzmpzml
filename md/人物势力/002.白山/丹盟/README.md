## 势力概况

白山地区的炼丹宗门联盟，以丹道起家。韩天青为丹盟元婴老祖，与灵木盟长期对峙。丹盟在白山大战中与灵木、离火盟作战，曾退守丹青山。

## 组织结构

- **老祖**: 韩天青（元婴）
- **金丹**: 韩平

## 势力关系

- 与灵木盟、离火盟敌对
- 与白山剑派、幻剑盟结盟

## 主要事迹

- 第16卷：韩天青参与丹青山之战
- 白山大战：与白山剑派、幻剑盟联手对抗灵木、离火联军
