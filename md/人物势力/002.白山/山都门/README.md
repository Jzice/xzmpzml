# 山都门

## 基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 山都门 |
| 类型 | 白山小宗门 |
| 所在 | 白山地区 |
| 首次出现 | 第4卷 |

## 势力概况

白山地区的小型宗门。

## 主要人物

- 斯温煜 - 练气后期修士
- 斯温泰 - 山都门修士

## 势力关系

- 与楚秦门有竞争关系

## 主要事迹

- 第4卷：斯温煜参加黑河坊擂台赛
- 斯温煜在第一轮比赛中败于练气二层的古吉，成为擂台上的一大冷门
- 山都门因斯温煜的败北而无颜再呆下去，灰溜溜离开

## 备注

白山小宗门之一。

## 势力范围归属

- **上级势力**: 白山
- **同级势力**: 白沙帮、灵药阁、多宝阁、器符盟
- **下级势力**: 无
