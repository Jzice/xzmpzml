## 势力概况

离火盟是白山五行盟之一，属火，擅长火系功法。离火盟是白山三盟（灵木、连水、离火）之一，参与白山内部的势力争夺。离火盟响应大周书院号召，参与白山深处开辟战争。离火盟与楚秦门多次冲突，在山都山大战中与楚秦门对抗。

## 势力范围归属

- **上级势力**: 白山、白山五行盟
- **同级势力**: 灵木盟、连水盟、锐金盟、厚土盟、丹盟、器符盟
- **下级势力**: 无
