## 势力概况

厚土盟是白山五行盟之一，属土，擅长防御土系功法。厚土盟是白山五行盟（灵木、连水、离火、锐金、厚土）中最晚加入的联盟，与其他四盟共同构成白山派旗下的五行联盟体系。厚土盟参与白山内部的势力争夺，响应大周书院号召参与开辟战争。

## 势力范围归属

- **上级势力**: 白山、白山五行盟
- **同级势力**: 灵木盟、连水盟、离火盟、锐金盟、丹盟、器符盟
- **下级势力**: 无

## 设定关联
