## 势力概况

连水盟是白山五行盟之一，属水，擅长水系功法。白山地区的修真势力联盟之一，参与白山内部的势力争夺。连水盟是白山六大势力联盟之一，第4卷擂台赛期间参与黑河坊争夺，与楚秦门有竞争关系。

## 势力范围归属

- **上级势力**: 白山、白山五行盟
- **同级势力**: 灵木盟、离火盟、锐金盟、厚土盟、丹盟、器符盟、白沙帮
- **下级势力**: 无
