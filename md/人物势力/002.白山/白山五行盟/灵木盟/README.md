## 势力概况

灵木盟是白山五行盟之一，属木，擅长灵植相关功法。灵木盟是白山三大联盟之一，与连水盟、离火盟并列，在白山北部有重大影响力。灵木盟响应大周书院号召，参与白山深处开辟战争，与蛮荒地带接壤的利益巨大。灵木盟与楚秦门多次冲突，柴冠曾参与围杀楚红裳，两年后在山都山再次与楚秦门对抗。

## 势力范围归属

- **上级势力**: 白山、白山五行盟
- **同级势力**: 连水盟、离火盟、锐金盟、厚土盟、丹盟、器符盟
- **下级势力**: 无
