# 血刀

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 血刀（高和同） |
| 性别 | 男 |
| 修为 | 血影邪修 |
| 本命 | 血引遁 |
| 势力 | 醒狮谷 |
| 首次出现 | 第15卷 |
| 结局 | 被楚秦门众人联手击杀 |

## 角色关系

- **仇人**: 展元（杀害展元）
- **敌人**: 齐休、楚秦门

## 主要事迹

### 杀害展元

- 血刀是杀害展元的凶手
- 多年前潜入黑河坊作乱
- 展元为了掩护弟子撤退而死
- 秦唯喻也在那次被鬼修杀害

### 醒狮谷藏身

- 血刀逃入醒狮谷
- 在血池附近修炼血影邪功
- 被楚秦门众人找到下落

### 楚秦门复仇

- 展仇坚持要为父报仇
- 齐休拗不过，带队进入醒狮谷
- 几经波折到达血池
- 众人联手与血刀血战
- 最终血刀被众人联手击杀

### 高和同身份

- 血刀真实身份是高和同
- 委托裴雯用秘法降临杀楚红裳
- 胆小不敢出齐云山一步

## 备注

- 血影邪修以血为媒
- 【血引遁】是独特逃命功法
- 醒狮谷是其藏身之地
