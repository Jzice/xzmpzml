## 势力概况

御兽门是**超级宗门**，位列**上八门**，被誉为**最强宗门之一**。其最强者为御兽门镇守，实力深不可测。

御兽门拥有独立发动**开辟战争**的特权，这是其无上地位的象征。在修真世界中，开辟战争涉及新世界的开辟与征服，唯有最强大的势力才具备此等能力。
