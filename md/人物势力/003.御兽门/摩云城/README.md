## 势力概况

摩云城是御兽门在南疆的重要附属势力，其城主为御兽门元婴修士。摩云城与御兽门总山保持密切联系，是御兽门在南疆的核心据点之一。

城主任摩云城主，修为元婴，本命灵兽为神目苍鹰，具有克制影遁的能力。在御兽门对楚无影的审讯行动中，摩云城发挥了重要作用。
