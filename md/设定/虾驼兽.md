# 虾驼兽

御兽门特产的一种专门用于挖矿的低阶灵兽。

## 基本信息

| 项目 | 内容 |
|------|------|
| 品阶 | 无品无阶 |
| 价格 | 半枚三阶灵石/只（第17卷第393章） |
| 产地 | 大漠中的某处御兽门飞地（非南疆御兽门统属） |
| 用途 | 专用于灵石矿的挖掘开采 |
| 寿命 | 绵长（数十年至数百年不等，远超低阶修士寿命） |
| 来源 | 仅御兽门有售，通过御兽门兽船运输配送 |
| 首次出现 | 第16卷第361章（南宫嫣然建议购买） |
| 首次实装 | 第17卷第393章（首批1000只交付） |

## 外形特征

**第17卷第393章（山都山杂事）原文描述：**

- 乘坐兽船抵达，兽船本体为一只极大的**飞蟹**（青黑蟹壳、巨大透明翅、腹部挂舱室）
- 远看极像人类，或者说像**瘦削的驼背老人**——楚秦门众人初见时"心里都一咯噔"
- 近看才能分辨：背部有**骆驼式的单峰**（高隆起），故得名"驼"
- 外壳呈**黄沙般的粗糙壳子**，面部为**虾类生物的面部**（凸出的双眼转动不停），故得名"虾"
- 退化了的须足等物贴在身前
- 有一双**虾螯像人类的双手**，走路时会前后摆动
- 用双脚走路
- 有一定灵智，能表现出**恐惧**情绪（"看着周遭环境和楚秦人们，分明表露出恐惧"）

## 驱使之法

1. **铃铛控制**：御兽门提供专用铃铛，摇响后虾驼兽的注意力会全部集中过来
2. **精神力沟通**：虾驼兽听不懂人话，必须用特殊的精神力波动与它们沟通
3. **驯如狗**：类似于训狗，摇铃铛即可指挥它们"坐下、跪、打个滚儿"
4. 御兽门修士当场现教，修士学起来不费什么劲（第17卷第393章）

## 饲养与维护

| 项目 | 说明 |
|------|------|
| 食水 | 很好养活，但**不要给它们喝太多的水** |
| 运输 | 通过御兽门兽船配送，舱室内可装上万只，一次性配送多家 |
| 损耗 | 死亡不包赔，"死也是死兽类，再买就是了" |
| 替代人力的优势 | 避免奴役凡人的恶名；死兽类比死人麻烦少 |
| 经济性 | 一只半枚三阶属于"十分贵了"，但挖矿效率远超凡人 |

## 主要用途

### 核心用途：灵石矿挖掘

虾驼兽是天生的挖矿生物——力大、夜视能力好、寿命不短、最重要的是听话。原文明确："天生的挖矿生物，力大，夜视能力好，寿命不短，最重要的是听话"（第17卷第393章）。御兽门推销语为"凡是大矿，都要用到我家这虾驼兽"。

在楚秦门的应用：
- **天引山矿**：首批1000只投入，替换凡人劳力（第17卷第393章）
- **山都山大矿**：取代凡人矿工，矿区"除了十来名家中修士做监工，只剩下些少量服侍仙师的凡人，其余干活的，全是虾驼兽"（第17卷第405章）
- **天引、山都二山**因战祸灵脉已毁，虾驼兽挖就挖了（第19卷）
- **丹盟谈判中的威慑工具**：齐休以"好多虾驼兽需要照看"为由催韩掌门速决（第19卷第481章），后运虾驼兽的飞梭中藏有剑阵，沙诺以此发动"万剑齐发"攻击（第19卷第484章）
- **碧湖宫/九星坊**：搬运虾驼兽的飞梭被用于伪装偷袭（第19卷第484章）

### 其他用途

- **天引寺的放养**：妙清送数千只虾驼兽给法引养在天引寺中（第24卷第644章）
- **轻罪囚犯的劳役**："下矿一年"的惩罚——不是让虾驼兽挖矿，而是让犯人给虾驼兽搬食物（第20卷第508章法引的惩罚方式）
- **佛门教化**：一只虾驼兽曾在天引寺外"学着大家盘膝跪下，双掌合什，口中喃喃有声，大虾般的兽脸上，好像还真显出了些许虔诚颜色"（第24卷第644章）
- **天引寺门卫**：一只虾驼兽"身穿僧袍，用大笤帚在门外扫着落叶"（第24卷第651章）

## 重要事件时间线

### 第16卷第361章（自创一功法）—— 提议
南宫嫣然向齐休建议从御兽门购买虾驼兽用于天引山挖矿，理由是"那样只用留部分人手御使，速度快还安全，反正死也是死兽类，再买就是了"。齐休批准，让南宫嫣然"你看着办吧"。

### 第17卷第393章（山都山杂事）—— 首批交付
御兽门飞蟹兽船送达第一批**1000只**虾驼兽。楚秦门众人初见时因其外形接近人类而惊叹。御兽门修士现场教授驱使方法（铃铛+精神力）。南宫嫣然自掏腰包垫付灵石（"我和长风的，先垫着吧"）。虾驼兽立即投入天引山矿井，让服劳役的凡人"响起压抑的欢呼声"。

### 第17卷第405章（后续的安排）—— 全面替代
矿区"除了十来名家中修士做监工，只剩下些少量服侍仙师的凡人，其余干活的，全是虾驼兽"。虾驼兽已完全替代凡人成为主要劳动力。

### 第19卷第467章（平淡的饮宴）—— 法引的善待引发抱怨
法引任楚秦盟巡察使期间严令不得虐待虾驼兽。北烈山熊氏和元和山宁氏修士抱怨法引"说什么要善待一切生灵，挖矿的虾驼兽都不能驱使太累"。山都大矿的灵石产量因此下降到往年的五分之一。但因楚秦门已手头宽裕，不指望矿场活命，齐休也没让法引滚蛋。

### 第19卷第479章（第一次协商）—— 效率下降
"因为楚秦盟巡察使法引和尚严令不得虐待虾驼兽的关系，山都山矿场的开采效率大幅下降。"矿场只有两三个低阶修士做监工，百无聊赖地盯着慢吞吞行动的虾驼兽们。

### 第19卷第481章（第二次协商）—— 谈判威慑
齐休与丹盟韩阎老谈判时，以"我那边好多虾驼兽需要照看"为由催促对方速决。此后虾驼兽运输飞梭被用作万剑阵的伪装——沙诺从运虾驼兽的飞梭中放出无数飞剑，"虾驼兽"只是障眼法（第19卷第484章）。

### 第19卷第485章（玉鹤终证道）—— 撤退
虾驼兽重新装船，"天引、山都二山那是遭了战祸，灵脉本就已毁乱的，挖了就挖了"。实际上齐休根本不会真挖丹盟的山门，虾驼兽只是谈判筹码。

### 第23卷第622章（山都之战上）—— 散修占领
山都山被白山散修趁乱占领时期，由于虾驼兽"只要获得了简单驱使之法，就可以让这些灵智低微的生物勤勤恳恳埋头干活"，占据者不急跑路，继续用虾驼兽挖灵石。古熔攻来时，从储物袋中拿出驱使之法交给古剑门门主。

### 第23卷第623章（山都之战中）—— 丹药事件
山崩时一些顶级丹药落入矿坑，"便宜了下面矿坑里聚成一团躲避山崩的虾驼兽，引起好一番争抢"。虾驼兽在躲避灾难时也有群体行为。

### 第24卷第644章（水到渠自成）—— 天引寺放养
离火之乱后，妙清"送来数千只本在山都山中挖矿的虾驼兽"给法引养在天引寺，理由是"不忍见这些生灵受楚秦日日驱使折磨"。虾驼兽的佛门修行开始，一只虾驼兽竟在殿外学着盘膝跪下、双掌合什、口中喃喃有声。

### 第24卷第651章（执念与因果）—— 穿僧袍的门卫
南宫嫣然到天引寺时，"奇怪瞥了眼身穿僧袍，用大笤帚在门外扫着落叶的一只虾驼兽"。她向法引索要那几千只虾驼兽，被法引以"乃受妙清檀越当年所托"为由拒绝。虾驼兽寿命"绵长"，法引俩徒弟都老了，虾驼兽还"好端端地放养在寺中"。

### 第24卷第653章（大道与本心）—— 再买提议
南宫嫣然被拒后气鼓鼓地回到山都山，说"没办法，只有再找御兽门买一些虾驼兽来了"。矿场执事问"若我们买来虾驼兽后再被她（妙清）看到的话"，南宫嫣然抱怨妙清"多事"。

## 文学意义与设定分析

虾驼兽是《修真门派掌门路》中最有特色的非战斗灵兽设定之一，具有多重叙事功能：

1. **人性化反思的载体**：虾驼兽的外形"远看太像人类了"（第17卷第393章）这一特征反复出现，使它们成为介于"工具"和"生灵"之间的模糊存在。法引对它们的善待与楚秦上下对它们的工具化使用形成张力。

2. **佛门慈悲理念的具象化**：妙清送虾驼兽给法引养、法引严令不得虐待、虾驼兽学僧人盘膝诵经——虾驼兽成为天引寺佛门理念的"非人类见证者"。一只虾驼兽穿僧袍扫地、学诵经的场景是全书中少有的"非人类修行"意象。

3. **矿区经济变化的风向标**：从最初南宫嫣然自掏腰包垫付（第17卷），到产量下降到五分之一后楚秦门仍不在意（第19卷），再到南宫嫣然再买被拒（第24卷）——虾驼兽的兴衰反映了楚秦门从穷困到富足再到宗门形态变化的全过程。

4. **战术伪装**：运虾驼兽的飞梭中藏剑阵（第19卷第484章），虾驼兽被用作战争中的障眼法，展现了楚秦门在资源匮乏时期的灵活战术。

5. **寿命对比的时间尺度**：虾驼兽寿命绵长，"法引俩徒弟老了，当年被押去山都山挖矿的那些灵木盟仙凡战俘，如今也老的老死的死，而那数千只虾驼兽还好端端地放养在寺中"（第24卷第651章）。这一对比提示了修行世界中的时间流逝——低阶修士一代代老去，而灵兽仍在。

## 关联角色

- **南宫嫣然**：楚秦门庶务掌门，虾驼兽的主要采购者和管理者
- **法引**（天引寺主持）：善待虾驼兽，禁止虐待，受妙清之托在天引寺放养
- **妙清**：原尼姑，将虾驼兽送离矿区至天引寺放养
- **齐休**：楚秦掌门，批准购买并用虾驼兽作谈判筹码
- **沙诺**：利用运虾驼兽飞梭隐藏剑阵发动攻击
- **古熔**：山都之战中缴获虾驼兽驱使之法
- **英伯**：丹盟客卿，不认得虾驼兽（说明此物在白山不常见）
- **韩阎老**：丹盟掌门，认得虾驼兽，被齐休以此威胁
