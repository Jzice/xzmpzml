## 第021卷 醒狮御兽与九星

齐休决定暂不合并楚秦盟，继续以盟形式存在。御兽门开始开辟醒狮谷，齐休传消息给小黑让它通知摩云鬣撤离，小黑则去通知了元犼醒狮，醒狮听后返回醒狮谷战斗，御兽门损失惨重。外海四位金丹接下暗杀齐休、沙诺等人的黑活，以此与楚秦门交易意图在白山谋一领地安定下来。不料楚秦门用他们来练手，灭杀于白沙滩，并意外得知外海有异变发生。展剑锋一行人于酆水之地遭遇鬼灾，三天后获救，而后酆水开辟战争结束。醒狮谷众古兽得醒狮通知后南迁，楚无影一同南迁。
