## 第023卷 乱世再起秦与罗

齐休金丹六层，秦光耀在南宫家助力下结丹，欲联合楚秦门右山秦氏、罗氏两家搅局为祸楚秦。楚红裳借莲藕重塑肉身。蔡渊透露楚红裳遇难背后缘由，齐休分析局势，提出楚家与南宫家和解。燕南行陨落，齐休探出罗家心思。楚家与南宫家正式和解。小魔渊开辟完成，沙诺成就金丹，楚问成功分封白塔城，建海楚门。齐休去黑河下空间裂缝寻楚神苍，发现全知神宫，神宫带齐休回顾了三千多年前的诛“墨蛟”之战，而后跟着“墨蛟”离开。齐休被困裂缝期间，柴艺、郎季高成功鼓动古熔去诓齐休到离火盟，然而只诓来顾叹。秦光耀、罗姿等人在楚秦南部山门攻城掠地。楚神通去碧湖宫求灯，归途遇离火、厚土、锐金三盟元婴埋伏，本不落下风，又遭白山剑派元婴偷袭，负伤逃回齐云。古熔率离火大军出击楚秦。秦长风迎战崔元青，胜。楚问传送来齐云，齐休由楚问带着在齐云奔走打听局势，而后回思过山。全知神宫与墨蛟现身死沼，天地峰化神与枯荣聂疯子先后交手，而后发现君旋山只剩贾长庚的傀儡之身。楚问率领的南楚大军与古熔率领的离火大军交会山都山，楚问率先与厚土、锐金与白山剑派三位元婴交手，斩灭厚土元婴法相，刺破锐金元婴法相，白山剑派元婴亦受伤，楚问也被卸下一臂，离火元婴与离火军阵出动，齐云楚家军阵也到达，楚问趁机逃回军阵。齐休为避免郎季高褫夺古熔权力，遣姚青等人于器符城边境散布消息为古熔造势并贬低郎季高。后楚秦军阵与郎季高所率离火军阵战于帕吉涧，楚秦且战且退，回至北烈山。郎季高部清扫战场后快速与山都山古熔部回合，姚青等人反过来夸大郎季高战果，楚秦大军则趁古与郎争权之际，从死沼绕行与山都双楚大军会合。双楚与楚秦三军和离火两万大军决战山都山，终赢下。
