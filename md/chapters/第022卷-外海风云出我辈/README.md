## 第022卷 外海风云出我辈

外海魔烟四起，齐云楚家、南楚门与楚秦计划参与外海光复战争及小魔渊开辟战争，为结婴后的楚问分封外海积攒功绩点。光复战争共分三路，楚秦与三楚大军于北路进发，意取北路的白塔城为日后分封地。南宫止为搏楚红裳好感，亦率三千六百人参战，但表白被拒。楚秦大军由顾叹执掌军权。北路各部先是各自为战收复小岛，在中路军收复淞平岛后，开始联合作战向小魔渊推进。鱼尾岛分兵之前，南宫止不断纠缠，不仅牵线南路外海宗门挖三楚及楚秦墙角，还插手三楚及楚秦一系继承争议，惹得三楚与楚秦苦不堪言。分兵时，南楚门先走一步，齐休则带兵去襄助中路史万奇部收复铁风群岛，不料楚红裳遇难，元婴离体逃回南楚门临时大营，齐休带着元婴之体的楚红裳寻玉鹤帮忙，终回到南楚门。
