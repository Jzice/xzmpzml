## 第015卷 醒狮谷里报冤仇

楚秦门众人得知杀害展元的凶手血影邪修“血刀”的下落，齐休拗不过展仇，一行人便一同进醒狮谷报仇。几经波折终于到达血刀所在地血池，展仇不幸陷落血池附近重土地，剩余众人联手灭杀血刀，齐休身受重伤。展仇于重土地底碰上了杀害秦唯喻的鬼修，借黑河珠救下唯喻灵魂，自己则与鬼修同归于尽。奈文霖救下重伤的齐休，自己却因吸入过多病血之气入魔，被魔道功法反噬，惨死在齐休面前。
