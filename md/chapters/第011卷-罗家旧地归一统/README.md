## 第011卷 罗家旧地归一统

楚秦门势力渐起，为自保领周边势力救楠笼山对上宁家，后又为避免奈文家灭宁家后独大，逼出敢家后领众人去救宁家而对上奈文家，跻身罗家旧地第三大势力，最后在宁家与奈文家合攻时，允利使对方九位筑基反水，终赢下此战，一统罗家旧地。
