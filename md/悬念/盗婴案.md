# 盗婴案

## 概述

盗婴案是贯穿全书的重要事件之一。南楚楚家在齐云楚家的指使下，长期从白山各地偷盗天赋绝佳的婴儿，由楚夺操办，齐休曾参与其中。此事涉及南楚、齐云楚家、以及背后的更高层势力，直接影响了楚希钰、楚无影等人的一生命运。

## 事件经过

### 楚夺操盘，齐休参与

盗婴行动由南楚楚家的楚夺主导。楚夺选中齐休参与盗婴，是因为当年楚红裳降临黑河峰时看中了齐休。齐休练气二层时即被楚红裳逼迫，为其盗婴做掩护。

> 第004卷：「为何楚夺会让他参与偷盗婴儿这种有伤天和之事？」

齐休具体参与的内容包括：
- **仙林坳掩护**：楚夺被追杀时，将怀中婴儿丢入正在湮灭的幻阵，齐休负责接应掩护
- **转移视线**：齐休故意在器符盟诸金丹面前演戏，将盗婴嫌疑引向山都魏家，使玉鹤白自背了罪名

> 第016卷：「楚夺盗婴是他向玉鹤告的密,但楚青玉不知内情,只当他叛门而出。毕竟是盗婴受害人,楚希钰甚至没把事情闹太大。」

### 盗婴的真相——楚无影并非盗婴

第013卷中，楚无影的真实身份被揭开——他根本不是当年楚夺告诉齐休的"白山盗婴之一"，而是正牌楚慧心嫡传后裔，南楚楚家真正血脉！

> 第013卷：「齐休颓然软倒，终于明白了，楚无影根本不是当年楚夺告诉自己的白山盗婴一员，而是正牌楚慧心嫡传后裔，南楚楚家真正血脉！好个楚夺！真真假假，骗得自己好苦！」

楚夺以"盗婴"为名向齐休隐瞒楚无影的真实身份，实际上是为了保护楚家嫡系血脉。

### 背后势力的逼迫

楚夺曾对齐休透露过核心隐秘：

> 第013卷：「我南楚楚家做下盗婴这种丧阴德之事，也是被人逼迫，根本不是为了一己之私，更没得到半点好处。」

那么盗婴的好处到底被谁得了？齐休推断：

> 第013卷：「难道南楚楚家盗婴，好处全给齐云楚家得去了？」

楚震一死，南楚楚家与齐云楚家面和心不和，楚夺才敢透露这等隐秘。结合楚家高层（如楚震）依赖天地峰座主的背景，盗婴案可能涉及天地峰一系的某种谋划。

## 关键人物

| 人物 | 角色 |
|------|------|
| **楚夺** | 盗婴案的主要执行者，南楚楚家的操盘手 |
| **齐休** | 参与掩护和转移视线，知情者 |
| **楚无影** | 表面上是盗婴之一，实则是南楚嫡系血脉 |
| **楚希钰** | 真正的盗婴受害者，后被御兽门收留 |
| **玉鹤** | 被冤枉为盗婴嫌犯，引出御兽门与白山的矛盾 |
| **楚红裳** | 幕后指使者之一，知晓内情 |
| **楚震** | 齐云楚家老祖，可能是真正的受益方 |

## 遗留悬念

1. **逼迫南楚盗婴的究竟是谁？** 楚夺说"也是被人逼迫"，但未明说是谁。可能是齐云楚家（楚震），也可能是天地峰座主一系更高层的力量。
2. **盗来的婴儿都去了哪里？** 除了已知的楚希钰等人外，其他被盗婴儿的下落不明。
3. **盗婴案与齐休身世是否有关？** 齐休也是弃婴，他是否也是某次盗婴行动的产物或牺牲品？
4. **楚慧心的诡代之法传承与盗婴案有何关联？** 楚慧心秘传是楚红裳笼络齐休参与盗婴的筹码，二者之间并非偶然。
