# 万骨

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 万骨（原名精细鬼） |
| 性别 | 男 |
| 修为 | 金丹（鬼修），前世为空间大修士 |
| 本命 | 小丹炉（残破） |
| 势力 | 散修/鬼修 |
| 首次出现 | 第015卷第339章（以"万骨"自称） |
| 结局 | 第023卷第609章被姜家元婴杀死于思过山 |

## 别名/称号

- **精细鬼**: 原名，3000年前在另一界时的称号
- **万骨**: 第608章在楚秦山自改之名（"以后，我便改名唤做'万骨'如何？"）
- **空间大修士**: 真实身份——来自另一界的空间属性大能

## 外形描述

- **鬼修形态**: 身形瘦削、声音苍老的鬼修，附着于一尊残破的小丹炉上
- **鬼蛛形态**: 曾以鬼蛛形态现身（第405章），寄身于残破丹炉之中
- **特征**: 声音苍老沙哑，擅长隐匿和探查

## 法宝与能力

### 前世（空间大修士）
- **空间神通**: 生前为空间属性大修士，精通空间类法术和禁制
- **空间困阵**: 其创造的空间曾困住墨蛟，与全知神宫相关

### 今生（鬼修）
- **探查能力**: 可潜入危险区域探查情报（如血井探秘）
- **鬼修法术**: 具备鬼修常见手段，能附物、隐匿
- **丹炉寄身**: 残破丹炉为其鬼修形态的寄托之物

## 主要事迹

### 前世：另一界的空间大修士

- 万骨本非此界之人，原是另一界的空间属性大修士（空间大能）
- 不知因何原因被齐云姜家元婴修士擒拿暗害
- 死后灵魂被姜家元婴封印于一只小丹炉中，困于姜云山废墟

### 精细鬼时期：与姜炎相遇（第339章）

- 第339章，姜炎在姜云山小丹炉中唤醒了精细鬼的魂魄
- 精细鬼从此附着于残破丹炉，成为姜炎唯一能交流对话的存在
- 姜炎常被认为在自言自语，实则在与精细鬼交谈
- 精细鬼对姜家有深仇大恨（因其前世即为姜家元婴所害），怂恿姜炎对抗姜家

### 醒狮谷冒险（第340-405章）

- 第340章，精细鬼潜入血井探查，回报所见惊人景象
- 第349-350章，精细鬼察觉灵木的异常，向姜炎报告蹊跷之处
- 第355章，精细鬼被派去参与挖矿
- 在此期间一直作为姜炎的搭档和军师，提供情报和建议

### 逃亡之路（第405-453章）

- 第405章，精细鬼以鬼蛛形态出现，与姜炎商议从山都山废弃矿洞逃走的计划
- 第427章，参与穿越死亡沼泽的逃亡路线
- 第430章，精细鬼自称"曾独掌一方世界"，后又矢口否认，称自己一直是鬼修——这是其真实身份的首次暗示
- 第453章，随何玉、姜炎利用北丁申山传送阵一起逃离

### 改名万骨（第608章）

- 第608章，在楚秦山上正式自改名为"万骨"，告别精细鬼之名
- 宣称"以后，我便改名唤做'万骨'如何？"
- 此名号既标志其鬼修身份，也隐含对前世陨落的纪念

### 死亡：思过山之战（第609章）

- 第609章，万骨在思过山被姜家元婴修士击杀
- 姜家元婴以居高临下之势出手，万骨不敌，魂飞魄散
- 死前万骨对姜家元婴充满恨意，因其前世即为同一位姜家元婴所害

### 死后揭秘：空间大能的真相（第610、654章）

- 第610章，万骨死后其所造空间露出端倪——他曾制造一个特殊空间困住了墨蛟，此空间与全知神宫有莫大关联
- 第654章，齐休得知万骨的真实身份：他原是另一界的空间大修士，被姜家元婴擒杀后灵魂被封于丹炉
- 其制造的空间之所以能困住墨蛟，正因其前世身为空间大能的底蕴
- 这一揭秘串联起多个伏笔：墨蛟被困之谜、全知神宫的关联、精细鬼的异常能力

## 势力范围归属

- **势力范围**: 散修鬼修，长期活动于姜云山、山都山、楚秦山一带
- **前世归属**: 另一界的空间大修士
- **相关势力**: 姜炎（合作）、姜家（仇敌）

## 设定关联

- **空间神通**: 万骨前世作为空间大修士的能力，是全书少有的空间属性修行者
- **全知神宫**: 其所造空间与全知神宫相关联，困住墨蛟
- **姜炎线**: 万骨是姜炎早期最重要的伙伴和引路人之一
- **鬼修设定**: 展示了高阶修士陨落后化为鬼修的可能性
- **跨世界设定**: 万骨来自另一界，为此界与其他世界的关联提供了线索

## 备注

1. 万骨是全书重要的"隐藏大能"角色，表面为普通鬼修，实为来历不凡的空间大修士
2. "精细鬼"是其在另一界时的原名
3. 第430章"曾独掌一方世界"的自语为后续身份揭秘埋下伏笔
4. 他制造的空间困住墨蛟是全知神宫线的关键环节之一
5. 被同一人（疑为姜焕）杀死两次（前世被杀封印，今世被灭魂），命运颇为讽刺
6. 是姜炎前期最为信任的伙伴，对姜炎的成长影响深远
