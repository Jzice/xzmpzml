# 楚秦门

## 基本信息


| 项目     | 内容                            |
| -------- | ------------------------------- |
| 名称     | 楚秦门                          |
| 类型     | 修真门派                        |
| 等级     | 金丹（开山老祖）→ 练气（当前） |
| 所在     | 原：齐云派范围 → 南疆          |
| 首次出现 | 第1卷                           |

## 势力概况

### 开山历史

- **开山老祖**: 秦烈儿（金丹修士）
- **师承**: 楚震（齐云派元婴长老）
- **命名**: "楚"字在前，为感念楚震恩情

### 宗门等级

- 老祖在时：可授予子爵
- 齐掌门时：连筑基都没有，只能授予士
- 三代之后，失去保护，进入弱肉强食时代

### 当前状况

- 29岁的齐休成为第五代掌门
- 面临三派（流云宗、荷花观、湟左詹家）吞并危机
- 在楚家帮助下，决定南迁南疆

## 组织结构

### 历代掌门

1. **秦烈儿**（开山老祖，金丹）
2. **秦德昭**（筑基中期，二代掌门）
3. **齐掌门**（练气大圆满，三代掌门）
4. **秦斯言**（练气后期，四代掌门，放弃）
5. **齐休**（练气二层，五代掌门）

### 主要人物

- 秦家嫡系：秦业、秦斯言、秦唯喻、秦师姐、秦长老
- 外姓弟子：齐休、齐掌门、张世石、展元、何玉、古吉、张师兄
- 四灵根弟子：黄和、虞景、沈昌、潘荣

## 势力关系

### 友好势力

- **齐云派楚家**: 楚震（师祖）、楚佑严（代表）
- **南楚门**: 未来重要盟友

### 敌对势力

- **流云宗**: 三派之一
- **荷花观**: 三派之一
- **湟左詹家**: 三派之一

## 主要事迹

- 开山立派，得楚震庇护数百年
- 齐掌门时期，门派衰落
- 三派入侵，楚秦门覆灭
- 楚佑严出面，安排南迁
- 齐休带领十名弟子，南迁南疆重新开始

## 备注

楚秦门从金丹大派的辉煌，衰落到练气小派，被迫南迁。是齐休成长的起点。

## 势力范围归属

- **上级势力**: 齐云楚家（原）→ 白山 → 最终独立
- **同级势力**: 三派（流云宗、荷花观、湟左詹家）
- **下级势力**: 楚秦门附庸（戚长胜、罗汉犇等）

## 位置

楚秦门最初位于[旧楚秦山](https://zml.huijiwiki.com/wiki/%E6%97%A7%E6%A5%9A%E7%A7%A6%E5%B1%B1 "旧楚秦山")，被三派攻破后迁往[黑河峰](https://zml.huijiwiki.com/wiki/%E9%BB%91%E6%B2%B3%E5%B3%B0 "黑河峰")。^[[1]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c15-1)^

在[天引宗之战](https://zml.huijiwiki.com/wiki/%E5%A4%A9%E5%BC%95%E5%AE%97%E4%B9%8B%E6%88%98 "天引宗之战")后，根据[楚夺](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E5%A4%BA "楚夺")的安排，迁至[仙林坳](https://zml.huijiwiki.com/wiki/%E4%BB%99%E6%9E%97%E5%9D%B3 "仙林坳")。^[[2]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c86-2)^

在[第七次天引山之战](https://zml.huijiwiki.com/wiki/%E7%AC%AC%E4%B8%83%E6%AC%A1%E5%A4%A9%E5%BC%95%E5%B1%B1%E4%B9%8B%E6%88%98 "第七次天引山之战")中临阵脱逃，迁至[九三坊](https://zml.huijiwiki.com/wiki/%E4%B9%9D%E4%B8%89%E5%9D%8A "九三坊")。^[[3]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c175-3)^

从[区豹](https://zml.huijiwiki.com/wiki/%E5%8C%BA%E8%B1%B9 "区豹")手中夺得[剑影山](https://zml.huijiwiki.com/wiki/%E5%89%91%E5%BD%B1%E5%B1%B1 "剑影山")，改名为[楚秦山](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E5%B1%B1 "楚秦山")。^[[4]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c180-4)^

齐休、齐妆结丹后，山门迁至[罗山](https://zml.huijiwiki.com/wiki/%E7%BD%97%E5%B1%B1 "罗山")，改名[思过山](https://zml.huijiwiki.com/wiki/%E6%80%9D%E8%BF%87%E5%B1%B1 "思过山")。^[[5]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c314-5)^

[第二次白山大战](https://zml.huijiwiki.com/wiki/%E7%AC%AC%E4%BA%8C%E6%AC%A1%E7%99%BD%E5%B1%B1%E5%A4%A7%E6%88%98 "第二次白山大战")中，占领了博木城，改名[楚恩城](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E6%81%A9%E5%9F%8E "楚恩城")。^[[6]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c643-6)^

南迁后，南迁初始团队的六家（潘荣的[潘家](https://zml.huijiwiki.com/index.php?title=%E6%BD%98%E5%AE%B6&action=edit&redlink=1 "潘家（页面不存在）")、虞景的[虞家](https://zml.huijiwiki.com/index.php?title=%E8%99%9E%E5%AE%B6&action=edit&redlink=1 "虞家（页面不存在）")、沈昌的[沈家](https://zml.huijiwiki.com/index.php?title=%E6%B2%88%E5%AE%B6&action=edit&redlink=1 "沈家（页面不存在）")、张世石的[张家](https://zml.huijiwiki.com/index.php?title=%E5%BC%A0%E5%AE%B6&action=edit&redlink=1 "张家（页面不存在）")、展元展仇的[展家](https://zml.huijiwiki.com/index.php?title=%E5%B1%95%E5%AE%B6&action=edit&redlink=1 "展家（页面不存在）")、秦唯喻的[黑河秦家](https://zml.huijiwiki.com/index.php?title=%E9%BB%91%E6%B2%B3%E7%A7%A6%E5%AE%B6&action=edit&redlink=1 "黑河秦家（页面不存在）")）被称为初始六家。

## 组织结构

### 掌门

* 第一任掌门[秦烈儿](https://zml.huijiwiki.com/wiki/%E7%A7%A6%E7%83%88%E5%84%BF "秦烈儿")，金丹修为。因在齐云派与众人不合而出走，并在楚震的帮助下开辟建立了楚秦门
* 第二任掌门[秦德昭](https://zml.huijiwiki.com/wiki/%E7%A7%A6%E5%BE%B7%E6%98%AD "秦德昭")，筑基中期修为。性格顽劣，喜欢欺压弱小。想给一位练气后期的韩姓修士教训，结果被反杀并洗劫。^[[13]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-:0-13)^

* 第四任掌门[秦斯言](https://zml.huijiwiki.com/wiki/%E7%A7%A6%E6%96%AF%E8%A8%80 "秦斯言")，金丹修为。因入赘[湟后安家](https://zml.huijiwiki.com/wiki/%E6%B9%9F%E5%90%8E%E5%AE%89%E5%AE%B6 "湟后安家")，将掌门之位传给齐休。

## 领民

最开始随楚秦门南迁的只有约两千人的秦家。^[[16]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-16)^

赵良德离开南疆前，托付了几十户共百来人，一起迁到了仙林坳。^[[17]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-17)^

此外再加上白家的三百来人，余家五十来人，在南迁第八年总人口将到两千五百左右。^[[18]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-18)^

在[兵站坊之战](https://zml.huijiwiki.com/wiki/%E5%85%B5%E7%AB%99%E5%9D%8A%E4%B9%8B%E6%88%98 "兵站坊之战")与[藏经阁之战](https://zml.huijiwiki.com/wiki/%E8%97%8F%E7%BB%8F%E9%98%81%E4%B9%8B%E6%88%98 "藏经阁之战")后，迁入莫家与阚家。^[[19]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-19)^

自南迁仙林坳以来，十多年间，随着人口的繁衍和迁徙，增了一倍，五千余人口中，右山秦氏最多，其后依次是左山秦氏，仙林秦氏，阚氏，白氏，明氏，赵氏，莫氏，黑河秦氏，余氏等等。^[[20]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c159-20)^

[散修之乱](https://zml.huijiwiki.com/wiki/%E6%95%A3%E4%BF%AE%E4%B9%8B%E4%B9%B1 "散修之乱")中，仙林坳附近的领民折损千余，仙林秦氏接近灭绝，阚家折损小半。之后有毛家近千人，和邝氏、冯氏近千人前来投奔。^[[21]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c167-21)^^[[22]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c176-22)^

在[第七次天引山之战](https://zml.huijiwiki.com/wiki/%E7%AC%AC%E4%B8%83%E6%AC%A1%E5%A4%A9%E5%BC%95%E5%B1%B1%E4%B9%8B%E6%88%98 "第七次天引山之战")后，楚秦门迁回南楚，但[左山秦家](https://zml.huijiwiki.com/index.php?title=%E5%B7%A6%E5%B1%B1%E7%A7%A6%E5%AE%B6&action=edit&redlink=1 "左山秦家（页面不存在）")、[赵家](https://zml.huijiwiki.com/wiki/%E8%B5%B5%E5%AE%B6 "赵家")、邝氏、冯氏与毛家的几百人，都决定留在仙林。^[[22]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c176-22)^

迁至[楚秦山](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E5%B1%B1 "楚秦山")后，领地人口八千余人，家族以人口排名分别是，右山秦氏，罗氏，毛氏，熊氏，柯氏，明氏，阚氏，闵氏，刀氏，莫氏，李氏，黑河秦氏，余氏，仙林秦氏，魏氏，张氏，多罗氏，骆氏，虞氏，沈氏，潘氏等。

[齐休81岁](https://zml.huijiwiki.com/wiki/%E6%8E%8C%E9%97%A8%E8%B7%AF%E5%A4%A7%E4%BA%8B%E5%B9%B4%E8%A1%A8#q81 "掌门路大事年表")时，领民已有近一万五千人，每年几乎都能保证起码有一位孩童入门。^[[23]](https://zml.huijiwiki.com/wiki/%E6%A5%9A%E7%A7%A6%E9%97%A8#cite_note-c240-23)^
