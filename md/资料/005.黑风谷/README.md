## 势力概况

黑风谷是此界人人喊打的外道宗门，源自已失传的万法门一脉。黑风谷拥有化神两人：创派祖师屠风，另一人十分低调神秘。其与天理门是死敌。

### 黑手组织

据【千修行述】记载，黑手组织便为黑风谷所有。黑风谷通过黑手组织渗透各派，行刺杀、情报、破坏之事。此信息最早见于楚震手稿。

### 与齐云的关系

第695章时，因白山突袭碧湖宫和黑山事件，齐云派和黑风谷已成互相需要之局。齐云派与黑风谷结盟，黑风谷的态度由敌视转为灵活外交。
