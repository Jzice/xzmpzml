## 势力概况

### 开山历史

- **开山老祖**: 秦烈儿（金丹修士）
- **师承**: 楚震（齐云派元婴长老）
- **命名**: "楚"字在前，为感念楚震恩情

### 宗门等级

- 老祖在时：可授予子爵
- 齐掌门时：连筑基都没有，只能授予士
- 三代之后，失去保护，进入弱肉强食时代

### 当前状况

- 29岁的齐休成为第五代掌门
- 面临三派（流云宗、荷花观、湟左詹家）吞并危机
- 在楚家帮助下，决定南迁南疆

## 组织结构

### 历代掌门

1. **秦烈儿**（开山老祖，金丹）
2. **秦德昭**（筑基中期，二代掌门）
3. **齐掌门**（练气大圆满，三代掌门）
4. **秦斯言**（练气后期，四代掌门，放弃）
5. **齐休**（练气二层，五代掌门）

### 主要人物

- 秦家嫡系：秦业、秦斯言、秦唯喻、秦师姐、秦长老
- 外姓弟子：齐休、齐掌门、张世石、展元、何玉、古吉、张师兄
- 四灵根弟子：黄和、虞景、沈昌、潘荣

## 主要事迹

- 开山立派，得楚震庇护数百年
- 齐掌门时期，门派衰落
- 三派入侵，楚秦门覆灭
- 楚佑严出面，安排南迁
- 齐休带领十名弟子，南迁南疆重新开始

## 备注

楚秦门从金丹大派的辉煌，衰落到练气小派，被迫南迁。是齐休成长的起点。

## 势力范围归属

- **上级势力**: 齐云楚家（原）→ 白山 → 最终独立
- **同级势力**: 三派（流云宗、荷花观、湟左詹家）
- **下级势力**: 楚秦门附庸（戚长胜、罗汉犇等）
