# 南宫止

## 基本信息

| 项目 | 内容 |
|------|------|
| 姓名 | 南宫止 |
| 性别 | 男 |
| 修为 | 元婴 |
| 本命 | 待查 |
| 势力 | 齐云派→南宫家 |
| 首次出现 | 第11卷（提及） |
| 现状 | 被软禁于齐云天地峰（截至第25卷） |

## 角色关系

### 所属
- **身份**: 南宫家元婴修士
- **家族**: 齐南城南宫家（拥有化神老祖的超级家族）
- **师承**: 无明确记载

### 相关
- **南宫木**: 南宫家化神老祖
- **楚红裳**: 南楚门掌门，南宫止曾追求
- **齐休**: 楚秦门掌门，曾欠南宫止人情
- **秦长风**: 南宫止曾试图联姻（嫁南宫家女子）
- **蓝隶**: 齐云天地峰元婴，负责看押南宫止
- **陆云子（陆崆）**: 齐云掌门

## 主要事迹

### 早期事迹

**帮助楚秦门**
- 楚秦门灭刘家时，南宫止给过一纸文书（第12卷）
- 避免了当地齐云修士插手，楚秦门欠他一人情
- 后楚秦门白赚一个筑基媳妇（南宫嫣然）进门

### 追求楚红裳

- 南宫止曾追求楚红裳多年（第13~14卷）
- 对楚红裳说："早几百年就和你说了，只要你答应和我携手，齐云山七阶秘境的五十年使用之权，就是我的聘礼……"
- 楚红裳对其厌恶，以话支开
- 南宫止也不纠缠，洒然一笑
- 此举却引得楚红裳对齐休迁怒发火（第14卷）

### 探宝黑河

- 召集齐休、栖蒙派元婴等到黑河峰探宝（第15卷）
- 嫁南宫嫣然给秦长风，拉齐休参与探宝
- 齐休因需请示楚红裳而拒绝直接下探
- 南宫止大笑表示若能把楚红裳找来则求之不得
- 结果齐休、楚红裳、南宫止、栖蒙派元婴等一行人全部被困于黑河峰底伪六道中数年之久
- 最后靠南宫家化神老祖南宫木出手，才得以逃脱生天

### 坐镇黑河坊，处置刑剑遇刺事件

- 刑剑在器符城被龙越云当作齐休误伤，伤势极重（第16卷）
- 南宫止因在黑河坊坐镇，最先赶到，以高阶丹药救治刑剑（丹药内竟有器灵虚体）
- 对灵木盟参与此事表示愤慨："这白山修士，太无法无天了"
- 其控制下的黑河坊禁止散修南下，灵木盟无法越过黑河坊增援
- 一度打算亲自带人增援白山，被族中其他元婴合力制止
- 龙家为避开他控制的黑河坊，绕道齐东城出外海

### 暗助齐休，赠予万兽识注

- 齐休进齐南城时，南宫止暗中传音，赠予一册【万兽识注】玉简（第17卷）
- 书中详细记载御兽门灵兽相生相克之秘，属齐云内部机密之物
- 对齐休在醒狮谷收服鬼蛟起了决定性作用——"只要鬼蛟一出，白虎俯首"
- 这帮忙帮得云淡风轻，但正中要害
- 南宫止为此需承担干系，事后故意回避签署中人协议

### 上门楚秦门，委托诱拐何玉

- 亲自出席楚秦门结丹典礼（第18卷第420章），给楚秦门壮声势
- 当众对南宫嫣然温言关怀，展现家族长辈姿态
- 私下委托齐休在稷下试炼时诱拐何玉——黑河峰底探宝的关键物事落在何玉手中
- 南宫家四十多年都拿何玉没办法，此为还南宫止人情的关键机会
- 齐休虽不愿，但南宫家族的意志和南宫止的人情，无力抗拒，必须报答

### 追捕姜炎与失陷酆水

- 青木之龙击破北丁申山后，组织南宫家人手封锁，意图顺传送阵追查何玉下落（第19卷）
- 加入追捕姜炎队伍：由南宫止（南宫家）、姬佳芊（归古派）、刑剑（执法峰）以及归儒派一位元婴组成
- 队伍通过传送阵来到酆水流域一带，途中遭遇古兽袭击，众人失陷
- 南宫木为救南宫止，联合归儒派化神在酆水发动开辟战争
- 开辟战争持续约二十年，最终救出南宫止等人（第21卷第524章）

### 参与碧湖宫事务，评议姜家分封

- 获救后返回齐南城，精神尚可（第21卷）
- 在齐南城遇到楚红裳，继续纠缠
- 碧湖秘境出事，与楚红裳一同前往处理
- 途中私下与齐休讨论姜家分封之事，评价姜明恪为"庸人"
- 暗示姜明恪上位对楚秦门可能是件好事
- 对齐休说："白山人都说你奸猾，此言非虚啊"

### 外海光复战争——率军参战

- 为搏楚红裳好感，独力招募三千六百名修士参战（第22卷）
- 与黑风谷元婴交涉试图拉拢合作，被楚问拒绝后也不强求
- 率军加入北路光复战争，意图为楚家助阵

### 海门岛表白被拒

- 在海门岛正式向楚红裳表白（第22卷第567章）
- 倾诉受困酆水时念念不忘，终于下定决心，以道心立誓
- 楚红裳三度后退，冷言拒绝："我都快听吐了"
- 一掌含怒而发，将南宫止震飞
- 南宫止仍不死心："红裳，我不会放弃的，我以道心立誓，终有一天，你会爱上我，嫁给我，与我携手大道，共攀永生。"
- 告辞出门时心绪波动，撞上齐休，元婴威压将其摔飞

### 拉拢楚秦门内部，埋下祸根

- 宴请秦长风、多罗森（两人都是南宫家女婿），大献殷勤（第22卷）
- 暗示秦长风该继承掌门之位而非顾叹："听人说齐休要将下任楚秦门主之位留给顾叹，我怎么记得该是你秦家的？"
- 允诺二人有事尽管来找他
- 私下得意自语："红裳啊红裳，你太小看我了，要想征服最烈的马，少不得准备些酷烈的手段，刚柔相济之下，我相信，你终将乖乖站回到我面前……"
- 利用秦光耀为内应，以结丹丹药引诱其叛门
- 不断纠缠楚红裳长达六年，牵线南路外海宗门挖三楚及楚秦墙角
- 插手三楚及楚秦一系继承争议，惹得三楚与楚秦苦不堪言

### 漆山岛之战

- 黑风谷、齐云楚家、南宫家联合攻打漆山岛（第22卷）
- 南宫止主动以楚秦门为前驱，对楚秦门极尽礼遇
- 称赞齐休为"人物"，两人合作尚算愉快

### 追逐楚红裳，引发软禁之祸

- 楚红裳遇难后，元婴离体逃回南楚门大营（第22卷）
- 南宫止违反军令，偷偷独赴楚红裳遇难处
- 寻不着后又用密法循迹追往铁风群岛，意欲半途劫人
- 实则被执法峰利用——齐休反其道而行之，先找玉鹤帮忙，终带楚红裳元婴逃脱
- 此事暴露后，南宫止成为众矢之的

### 被软禁天地峰

- 楚红裳遭险后真相大白，在三楚压力下被大周书院软禁（第23卷）
- 三楚以"南宫止欲得楚红裳已久，有人暗示后他去劫人"为由，顶住南宫木压力将其软禁
- 蓝隶奉命将南宫止从大周书院领回，暂时安排在天地峰内居住
- 被软禁后虽落魄，仍带讥讽："神仙打架，小鬼遭殃，当年在酆水被困时我就习惯了"
- 听说不知要被关到什么时候，面色立变
- 蓝隶传话："黑脸红脸白脸，你却是个灰脸"，南宫止听罢无言
- 秦光耀因失去庇护，骑虎难下，最终叛门投裴家
- 齐休分析局势时提及南宫止被囚对楚秦门局势的影响（第23卷）
- 楚问与蔡渊争论南宫止事件真相——蔡渊透露"有人暗示他"，指控铁风群岛有更高层面人物在等楚红裳
- 齐休推导执法峰利用南宫止的完整布局——执法峰用暗示手法让南宫止去劫人
- 秦光耀在南宫止被囚后用其许诺的结丹圣药结丹
- 后南宫家与三楚谈和，但南宫止仍被软禁多年

### 第24卷：一心大道向元婴

- 仍被软禁于天地峰
- 楚红裳与齐休讨论南宫止"有些冤枉"，补全执法峰整套计划——楚红裳说"南宫止搞出那许多事，所罚无非是受天地峰一系软禁，又不耽误修行，还便宜他了呢"
- 楚红裳出关后对南宫家仍有疙瘩，因南宫止之事与南宫梦产生不快
- 万兽识注仍被齐休使用（南宫止早年的赠予）

### 第25卷：美人恩重元婴成

- 仍被软禁于齐云山中多年——南宫晃提及"族中深孚众望的元婴修士南宫止因为外海和楚红裳那档子事，已被软禁于齐云山中多年"
- 秦长风与南宫家因南宫止引诱秦光耀叛门之事积怨颇深
- 楚红裳的不灭之体暴露后被担忧会引来"南宫止一类有心人觊觎"——南宫止已成为参考案例

### 与其他势力的关系

- 风度翩翩，行事洒落
- 与栖蒙派元婴很熟
- 家世显赫，即便修为不如对方，依然能做主
- 在齐云各势力中人脉广泛
- 对楚红裳痴情至深，但手段过激，终酿大祸
