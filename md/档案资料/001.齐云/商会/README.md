## 势力概况

齐云商会是齐云派旗下的商业联盟组织，整合了齐云势力范围内的多家修真商铺和交易机构。作为齐云派在经济领域的重要触角，齐云商会控制着大量的修真资源贸易，与白山地区的各修真商会既有合作也有竞争。商会旗下拥有多宝阁、灵药阁等主要分支，分布于白山、齐云城等地，从事法器、灵药等修真物资的买卖交易。

## 主要分支

- [万宝阁](万宝阁.md) - 法器交易
- [灵药阁](灵药阁.md) - 灵药交易
- [广汇阁](广汇阁.md) - 综合商会（高家）

## 势力范围归属

- **上级势力**: 齐云派
- **同级势力**: 南楚门、安家、姜家等齐云附属势力
- **下级势力**: 多宝阁、灵药阁
