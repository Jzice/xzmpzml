## 势力概况

锐金盟是白山五行盟之一，属金，专注于金属性灵材和炼器相关业务。锐金盟是白山三盟（灵木、连水、离火）之一，后扩展为白山五行盟（加入锐金盟、厚土盟）。锐金盟参与白山内部的势力争夺，响应大周书院号召参与开辟战争。第23卷中，锐金盟金丹修士崔元青败于秦长风。

## 势力范围归属

- **上级势力**: 白山、白山五行盟
- **同级势力**: 灵木盟、连水盟、离火盟、厚土盟、丹盟、器符盟
- **下级势力**: 无

## 设定关联

- 开辟战争
- 山都山大战
- 楚秦门冲突
