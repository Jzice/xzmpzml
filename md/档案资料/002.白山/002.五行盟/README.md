## 势力概况

白山五行盟是白山派下的五个金丹宗门组成的联盟，分别对应五行：灵木盟（属木）、连水盟（属水）、离火盟（属火）、锐金盟（属金）、厚土盟（属土）。五盟在白山区域各自拥有势力范围，共同构成白山派旗下的重要联盟体系。白山五行盟响应大周书院号召，参与白山深处的开辟战争，与蛮荒地带接壤的利益巨大。各盟之间既有合作也有竞争，在白山内部的势力争夺中扮演重要角色。
