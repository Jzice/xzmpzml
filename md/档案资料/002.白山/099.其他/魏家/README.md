## 势力概况

魏家原为北方小宗门，由魏玄主持，后迁移到白山。魏玄是金丹修士，与魏同（御兽门金丹老祖）为远房亲族。魏同借助御兽门的势力，召集各路修士灭了山都门，将地让给魏玄作为新的宗门所在。魏同出力，魏玄出名，两家并一家。
